# Blues' Hamburgueria API

API developed using Nest.js

## How to clone app

```bash
$ git clone git@github.com/mvpbarbosa/blue-hambluergueria-nest.git
```

## How to install dependencies

```bash
$ npm install
```

## How to run the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
